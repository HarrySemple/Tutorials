    Problems from week 4, due 1pm Tuesday 29th October.

    Ex 13. Question 4.

    Consider the function \(f(x, y) = 3xe^y - x^3 - e^{3y}\)
    1. (H) Show that \(f\) has only one critical point, and that it is a local maximum.
      Solution.
      \begin{align*} f_x(x, y) \amp = 3e^y - 3x^2, \\ f_y(x, y) \amp = 3xe^y - 3e^{3y}. \end{align*}
      Setting derivatives to zero we have
      \begin{equation*} 3e^y - 3x^2 = 0 \implies x^2 = e^y \implies x^2 > 0, \end{equation*}
      \begin{align*} 3xe^y - 3e^{3y} = 0 \amp \implies xe^y = e^{3y} = (e^y)^3 \\ \amp \implies x^3 = x^6 \\ \amp \implies x = 1, y = \ln(x^2) = 2\ln(1) = 0. \end{align*}
      Hence there is a single critical point, at \((1,0)\text{.}\) To determine the nature of this critical point we will use the second partials test, so we need the second order partial derivatives:
      \begin{align*} f_{xx}(x, y) \amp = -6x, \\ f_{yy}(x, y) \amp = 3xe^y - 9e^{3y}, \\ f_{xy}(x, y) \amp = 3e^y. \end{align*}
      Hence the determinant is
      \begin{equation*} D = - 6x(3xe^y - 9e^{3y}) - 9e^{2y}. \end{equation*}
      At the point \((1,0)\text{,}\) this is
      \begin{equation*} (-6)\times(3 - 9) - 9 = 27 > 0, \end{equation*}
      so \((1,0)\) is a local extremum. Now, \(f_{xx}(1,0)= - 6 \text{,}\) so it is a local maximum.
    2. (H) Show that this local maximum is not a global maximum for \(f\text{.}\)
      Solution.
      The value of \(f\) at this local maximum is \(f(1,0)=3−1−1=1 \text{.}\) So, to show that this is not an global maximum, we just need to find \((x,y)\) such that \(f(x,y)>f(1,0)=1\text{.}\) There are infinitely many points where this is true, but randomly trying values of \(x\) and \(y\) is not an efficient method of finding one. Here is the reasoning I used: the local maximum lies on the line \(y=0\text{,}\) where \(f(x,0)=3x−x^3−1\text{.}\) If we pick a negative value of \(x\) with a large modulus, then the \(-x^3\) that appears in this expression with be a large positive number, giving a value of \(f(x,y)\) greater than \(1\text{.}\) So, for example, try
      \begin{equation*} f(-10, 0) = -30 - (-1000) - 1 = 969. \end{equation*}
      Thus \(f(-10, 0) = 969 > 1 = f(1, 0)\text{,}\) the local maximum at \((1,0)\) is not a global maximum.
    3. Use Geogebra 3D
       3 
      mat.geogebra.org/3d?lang=en-GB
      or Desmos 3D
       4 
      www.desmos.com/3d
      to plot the surface defined by \(f(x,y)\) and check your answers in (b).

    Ex 14. Question 5.

    Consider the function \(f(x, y) = x^4 + y^4\)
    1. Use Geogebra3D to plot the surface defined by \(f(x,y)\text{.}\)
    2. (H) Find the critical point of \(f\text{.}\) Show that the second partials test to determine whether the crtical point is a local maximum, local minimum, or saddle point is inconclusive. Explain briefly why you think the test is inconclusive for this particular critical point.
      Solution.
      We have \(f_x(x, y) = 4x^3\text{,}\) \(f_y(x, y) = 4y^3\text{.}\) These are only equal to \(0\) at the point \((0,0)\text{,}\) so this is the only critical point of \(f\text{.}\) To (attempt to) use the second partials test, we calculate the second order partial derivatives:
      \begin{gather*} f_{xx}(x, y) = 12x^2, \\ f_{yy}(x, y) = 12y^2, \\ f_{xy}(x, y) = 0. \end{gather*}
      Hence the determinant is \(D = 144x^2y^2\text{.}\) At \((0,0)\text{,}\) this is \(D=0\text{,}\) so the second partials test is inconclusive. It fails because the orders of the terms in the original function are so high - each term has order 4, so must be differentiated four times to give anything non-zero at \((0,0)\text{.}\)
    3. (H) Show that the critical point you found in part (a) is a global minimum.
      Solution.
      Fist, observe that \(f(0,0) = 0\text{.}\) For all \(x\) and \(y\text{,}\) we have \(x^4 \geq 0\) and \(y^4 \geq 0\text{,}\) with equality only at \(0\text{.}\) Thus, for \(x\neq 0\) or \(y\neq 0\) we have \(f(x,y)>0=f(0,0)\text{,}\) so \((0,0)\) is a global minimum.
    4. (H) Find or construct an example of a function with a local maximum that the second partials test cannot detect.
      Solution.
      There are many examples. One simple one is \(g(x, y) = -x^4 - y^4\text{.}\) Then \(g\) has a local maximum (in fact, an global maximum) at \((0,0)\) that the second partials test cannot detect.
    5. (H) Find or construct an example of a function with a saddle point that the second partials test cannot detect.
      Solution.
      \(f\text{,}\)\(h(x, y) = x^4 - y^4\text{.}\)\(h\)\((0,0)\)

    Ex 15. Question 6.

    (H) A function \(f(x,y)\) is said to be harmonic if it satisfies Laplace’s equation:
    \begin{equation*} \Delta f := \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2} = 0. \end{equation*}
    Harmonic functions are used to model various physical quantities, including electric and magnetic fields, fluid flows, and steady-state temperature distributions.
    Determine whether the following functions are harmonic:
    1. \(\displaystyle f(x, y) = e^{kx}\cos(ky)\)
      Solution.
      \(f(x, y) = e^{kx}\cos(ky)\text{,}\) so
      \begin{gather*} f_x(x, y) = ke^{kx}\cos(ky), \\ f_y(x, y) = -ke^{kx}\sin(ky), \\ f_{xx}(x, y) = k^2e^{kx}\cos(ky), \\ f_{yy}(x, y) = -k^2e^{kx}\cos(ky). \end{gather*}
      Hence
      \begin{equation*} \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2} = k^2e^{kx}\cos(ky) - k^2e^{kx}\cos(ky) = 0, \end{equation*}
      so \(f\) is harmonic.
    2. \(\displaystyle f(x, y) = \ln(x^2 + y^2) + 2\arctan(y/x)\)
      Solution.
      Both \(\ln(x^2 + y^2)\) and \(2\arctan(y/x)\) are harmonic, so their sum is also harmonic.
