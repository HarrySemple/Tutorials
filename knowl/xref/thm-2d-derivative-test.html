Claim 14. Second Partial Derivatives Test (Theorem 2.8).

Let \(f\) be a function of two variables with continuous second-order partial derivatives in some disc centred at a critical point \((x_0,y_0)\text{,}\) and let
\begin{equation*} D = f_{xx}(x_0, y_0)f_{yy}(x_0, y_0) - f_{xy}(x_0, y_0)^2. \end{equation*}
  1. If \(D>0\) and \(f_{xx}(x_0,y_0)>0\) then \((x_0,y_0)\) is a local minimum.
  2. If \(D>0\) and \(f_{xx}(x_0,y_0)\lt 0\) then \((x_0,y_0)\) is a local maximum.
  3. If \(D\lt 0\) then \((x_0,y_0)\) is a saddle point.
